# 🇯🇵 日本旅遊 AI 智慧收據記帳系統

一個完全架設在 GitHub Pages 上的 Serverless 記帳神器！出國帶回的大量日本收據（全家、小七、羅森、LoFT 等），直接**批量拖曳上傳**，AI 就會自動辨識時間、日期、地點、將品項精準翻譯成**繁體中文**並將金額獨立分欄，最後一鍵匯出成完美相容 Excel、絕不亂碼的 CSV 記帳表！

## 🔑 核心教學：如何免費申請 Gemini API Key？

本系統需要自備 Google Gemini API Key 才能運作。Google 官方目前提供**每天 1,500 次的完全免費辨識額度**，一般旅遊記帳絕對綽綽有餘！請按照以下步驟免費申請：

### 第一步：前往 Google AI Studio
1. 點擊進入 [Google AI Studio 官方網站](https://aistudio.google.com/)。
2. 使用您的 **Google 帳號** 登入。

### 第二步：建立 API 金鑰 (Create API Key)
1. 登入後，點擊左上角的 **"Get API key"**（獲取 API 金鑰）按鈕。
2. 點擊藍色的 **"Create API key"**。
3. 在彈出視窗中，選擇 **"Create API key in new project"**（在新專案中建立金鑰）。
4. 系統會自動生成一串長程式碼（金鑰），點擊 **"Copy"** 將其複製並妥善保存。
   > ⚠️ **安全提示：** 請勿將您的 API Key 直接寫死在程式碼中或公開分享給他人！

## 💡 專業省錢與優化小技巧

如果你選擇使用 **付費模式** 享受超音速對帳，但又希望把帳單費用壓到最低，請務必這樣做：

1. **四周空白剪乾淨：** 日本收據通常是長條型，拍攝時請**將收據左右兩側不相干的背景裁切掉**，只留下有字的白色收據本體。
2. **降低照片解析度：** Google API 視覺計費是將高解析度圖片切成數個 $512 \times 512$ 的網格（Tiles）計費。稍微壓縮照片或用 LINE 傳送過再下載（利用 LINE 的自動壓縮），可以讓網格消耗量大幅下降，**費用直接打對折**！
3. **日常對帳切回免費：** 如果當天只有 3、5 張收據想順手記帳，記得在網頁上方一鍵切回**免費模式**，就能繼續安心白嫖 Google 的每日免費大禮包囉！
